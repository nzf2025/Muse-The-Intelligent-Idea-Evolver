# Muse: The Intelligent Idea Evolver 🧠✨

**Muse** is a modern, AI-driven cognitive companion designed to capture your fleeting "sparks" and transform them into structured, actionable insights. Inspired by the minimalist aesthetic of Obsidian, Muse acts as a world-class interviewer that challenges your initial thoughts to help you reach deeper understanding.

## ✨ Key Features

- 🎙️ **Multi-modal Capture**: Instantly capture inspirations via text, voice memos, or snapshots.
- 🧠 **AI-Powered Dialogue**: Engage in deep, interactive interviews with "Muse" to refine your raw thoughts.
- 📄 **Insight Synthesis**: Automatically transform your conversations into structured artifacts like **Deep Articles**, **Action Plans**, or **Interview Transcripts**.
- 🌑 **Obsidian-Style UI**: A focused, dark-mode interface designed for deep thinking and minimal distraction.
- 📱 **PWA Ready**: Install Muse on your mobile device for a seamless, app-like experience.
- 🔒 **Privacy Focused**: All sessions are stored locally, keeping your intellectual property under your control.

## 🛠️ Tech Stack

- **Frontend**: React 19 + TypeScript + Tailwind CSS
- **AI Engine**: Google Gemini 3 Flash (via @google/genai)
- **State Management**: React Hooks + LocalStorage
- **Routing**: React Router 7
- **Icons**: Font Awesome 6 + Lucide

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/muse.git
   cd muse
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env` file and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_key_here
   ```

4. **Run the app**:
   ```bash
   npm run dev
   ```
